# Unemployment Analysis with Python

This project is part of the Oasis Infobyte Data Science Internship (Task 2).

## Objective
To analyze unemployment trends using Python and data visualization techniques.

## Dataset
A sample unemployment dataset representing different Indian regions was used for analysis.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Analysis
- Average unemployment rate by region
- Unemployment trends over time
- Correlation between unemployment-related factors

## Conclusion
The analysis provides useful insights into unemployment patterns and trends.
